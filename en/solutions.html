Products

Cryptomator

Cryptomator is a cloud-optimized encryption software that is easy to use and open-source. The software encrypts files on a user's device before they are uploaded to a cloud storage or server. Cryptomator is used by companies, research institutes, journalists, and private users to protect sensitive data from unauthorized access by third parties.

This combination of transparency and simplicity solves the trust issues that users and businesses face when using cloud solutions. Cryptomator won the CeBIT Innovation Award 2016 for Usable Security and Privacy. Cryptomator is available for Windows, macOS, Linux, iOS, and Android.

Defendor

Defendor is a server software that provides secure workspaces for your network drives. The software adds security features, such as anti-malware filters, file encryption, file versioning, and backups, to your corporate cloud- or network-based file storages.
